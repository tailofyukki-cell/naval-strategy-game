## 🎮 ゲーム概要

「蒼海の覇者」は、提督の決断のような海軍戦略シミュレーションの雰囲気を持つオリジナルWebゲームです。
プレイヤーは**連合艦隊**を率い、**帝国艦隊**と戦いながら海域の港を占領し、蒼海の覇権を目指します。

- **ジャンル**: ターン制海軍戦略シミュレーション
- **プラットフォーム**: Webブラウザ（モダンブラウザ対応）
- **技術**: 静的サイト（HTML/CSS/JavaScript + Canvas）

---

## 🎯 勝利条件・敗北条件

### 勝利
- **港4つ確保**: 30ターン以内に4つ以上の港を占領する
- **完全制圧**: 敵の港を全て占領する

### 敗北
- 自軍の港が全て失われる
- 自軍艦隊が全滅する
- 30ターン到達時に港数で劣勢

## 🗺️ 操作方法

1. **艦隊選択**: マップ上の自軍艦隊（青色）をクリック
2. **移動**: 青くハイライトされたマスをクリックして移動
3. **戦闘**: 赤くハイライトされた敵艦隊マスをクリックすると自動戦闘
4. **占領**: 敵・中立の港（黄色ハイライト）に移動すると自動占領
5. **建造**: 自軍の港をクリック → 「艦隊建造」ボタン → 艦種を選択
6. **ターン終了**: 「ターン終了」ボタンで敵ターンへ

## 🗺️ マップ構成

12×8グリッドのマップに6つの港が配置されています：

| 港名 | 初期勢力 | 収入（燃料/鋼材） |
|------|---------|-----------------|
| アルバ港 | 連合艦隊（プレイヤー） | +10 / +8 |
| サウス基地 | 連合艦隊（プレイヤー） | +8 / +6 |
| ノルド港 | 帝国艦隊（敵） | +10 / +8 |
| イースト要塞 | 帝国艦隊（敵） | +8 / +6 |
| 中央泊地 | 中立 | +12 / +10 |
| 西方海峡港 | 中立 | +10 / +8 |

## 💰 資源システム

- **燃料・鋼材**の2種類の資源を管理
- 毎ターン開始時、保有する港から収入を獲得
- 資源を消費して港で艦隊を建造

---

## ⚔️ 戦闘システム

戦闘はオート解決（数式ベース）：

```
ダメージ = 攻撃力 × (1 ± 0.3のランダム係数) - 防御力（最低1）
```

- 移動力が高い艦隊が先攻
- 交互に攻撃し、HPが0になると撃沈
- 最大10ラウンド後はHP多い方の判定勝利

---

## 🤖 敵AI行動

敵AIは以下の優先順位で行動します：

1. 移動範囲内の港（プレイヤー港・中立港）を占領
2. 勝率60%以上の敵艦隊に攻撃
3. 最も近い未占領港へ移動
4. 資源が十分なら港で艦隊建造
